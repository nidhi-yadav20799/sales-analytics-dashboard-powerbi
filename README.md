# Sales Analytics Dashboard

This project is an interactive Sales Analytics Dashboard created using Power BI.

## Features
- Interactive Date Slicer
- KPI Card for Total Sales
- Customer Sales Distribution
- Top Customers by Sales
- Clean and modern dashboard layout

## Tools Used
- Power BI
- DAX
- Data Visualization
